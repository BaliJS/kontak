# Cara Install

- Clone project
- Masuk ke directory project
```
composer install
npm install
gulp
php artisan migrate
php artisan db:seed
php artisan serve 
```

Kalau perlu watch setiap perubahan file gunakan
```
gulp watch
```

# Library
- [HackCss](http://hackcss.com/)
- [VueJS](http://vuejs.org/)
